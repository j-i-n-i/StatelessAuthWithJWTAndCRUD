# StatelessAuthWithJWTAndCRUD
Stateless authentication using JWT with CRUD operations for a students table (Mongoose & Express). Frontend built with HTML, CSS, JavaScript/jQuery. Includes admin interface for registration, login, token generation, and API calls with Postman.
